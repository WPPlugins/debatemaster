== Description ==

Enables a shortcode, [debate], that allows multiple blog contributors to add their contributions to a discussion in an individualized way by displaying a custom background color, name, and a gravatar. I appreciate your feedback!

== Installation ==

Unzip debatemaster.zip into /wp-content/plugins/debatemaster/ .

Check out the DebateMaster options tab and set the colors for each contributor. That's about it.
